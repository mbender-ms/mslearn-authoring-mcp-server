# cross-post

Do not use.  

**Guidelines**  

- Although *cross-post* does appear in help documentation, do not use.  
- Instead, use "post in multiple channels."  

**See also** [post in multiple channels](~/teams-style-guide/a-z-word-list/p/post-in-multiple-channels.md)

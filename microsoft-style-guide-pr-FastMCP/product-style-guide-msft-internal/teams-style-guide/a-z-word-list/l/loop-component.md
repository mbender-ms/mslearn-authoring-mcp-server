# Loop component

A live, connected, actionable experience that allows the user to complete their work in the context of chat, emails, meetings, documents, and more. 

Examples of components are checklist, table, numbered list, paragraph.  

**Guidelines**  
- First mention: Loop component  
- Subsequent mention: component  
- Common phrasing around use in Teams messaging is "send a component," "insert a component into a message," and "edit a component."  

**Note**: *component* is singular.

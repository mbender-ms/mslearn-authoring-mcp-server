# extensibility

Do not use.  

The concept of extending Teams functionality with apps.  

**Guidelines**  
- For a technical audience only.  
- For end users, talk about using the apps in teams.  

**See also** [platform](~/teams-style-guide/a-z-word-list/p/platform.md)

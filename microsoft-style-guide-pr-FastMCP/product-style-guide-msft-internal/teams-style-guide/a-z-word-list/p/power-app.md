# power app


Do not use.  

A low-code app built with Microsoft Power Apps.  

**Guidelines**  

- Use *app* instead.  
- If you need to clarify what kind of app it is, you can say "an app built with Microsoft Power Apps" or "an app built on the Microsoft Power Platform."

# simultaneously ring

A call handling setting for incoming calls. 

When you receive a call, it rings another person or group of your choosing at the same time. 

The label for this feature in settings is "Also ring."

**Guidelines**

- All mentions: simultaneously ring
- Don't use "simulring."

# DirectSR

DirectSR is an API that provides Super Resolution (SR) support for D3D12 applications. This technique can increase performance, reduce strain on the graphics card, prolong battery life while gaming, and reduce overall power usage.  

**Guidelines**  

- All mentions: DirectSR  
- Don't localize

# identity services

The category of Azure services that includes Azure Active Directory.

**Guidelines**

Don't use *Azure Identity*, *Identity Services*, or similar phrases that imply that the category is a specific, named service. Instead, use the term *identity services* generically to refer to the group of services.Â

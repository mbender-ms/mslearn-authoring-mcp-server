# 64-bit and 32-bit

Use *64-bit version* and *32-bit version.* Don't use *edition* with these terms.

**Do**  
64-bit version of Windows  
32-bit version of Windows

**Don't**  
64-bit Windows  
64-bit edition  
32-bit edition of Windows

# Windows Autopatch

Microsoft Windows Autopatch is a service that automatically keeps Windows and Microsoft 365 apps up to date on enrolled endpoints.

**Guidelines**

- First mention: Microsoft Windows Autopatch
- Subsequent mentions: Autopatch

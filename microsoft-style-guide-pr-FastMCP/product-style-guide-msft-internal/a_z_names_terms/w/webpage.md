# webpage

**Guidelines**

One word, lowercase.

If the context makes it clear that you're referring to a webpage, it's OK to use *page* to refer to webpages.

**See also** [page](~\a_z_names_terms\p\page.md), [web](~\a_z_names_terms\w\web.md)

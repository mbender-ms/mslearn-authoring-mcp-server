# WAV, .wav

A digital media file format for storing sound.

The only acceptable uses are *.wav file* and *WAV format.*

Don't use:

- wave file
- wave format
- wave sound
- wav sound
- .wav sound

**Examples**  
You can save a sound file in WAV format.  
You can save an audio file with a .wav extension.
